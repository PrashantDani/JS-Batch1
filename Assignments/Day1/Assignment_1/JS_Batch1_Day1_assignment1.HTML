Question 1 :

Explore and explain the various methods in console function

1. Console.log() : Mainly used to log(print) the output to the console.

2. Console.error() : Mainly used to log(print) the error message.

3. Console.warn() : Mainly used to log(print) the warning message to the console.

4. Console.table() : used to generate a table inside the console.

5. Console.clear() : Used to clear the console.
